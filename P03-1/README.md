# • 정수형 분수 추상 자료형 (Fraction ADT) •

이 프로젝트는 정수형 분자와 분모를 이용한 분수 계산을 처리하는 추상 자료형(ADT)을 정의하고 구현합니다.

## 1. 개요
분수를 두 개의 정수가 아닌 구조체를 이용하여 하나의 자료형으로 취급하여, 유리수 계산을 논리적으로 처리하기 위해 설계되었습니다.

## 2. 데이터 구조
분수 자료형은 다음 두 개의 정수 데이터를 가집니다.
- **분자 (Numerator):** 정수형 (Integer)
- **분모 (Denominator):** 정수형 (Integer, 0이 아닌 값)

## 3. 연산 정의
분수 ADT는 다음과 같은 핵심 연산을 제공합니다.

| 연산자 | 기능 설명 |
| :--- | :--- |
| **Create** | 분자와 분모를 입력받아 새로운 분수 객체를 생성합니다. (분모가 0인 경우 예외 처리 포함) |
| **Add** | 두 분수를 더하여 그 결과를 반환합니다. |
| **Subtract** | 두 분수를 빼서 그 결과를 반환합니다. |
| **Multiply** | 두 분수를 곱하여 그 결과를 반환합니다. |
| **Divide** | 한 분수를 다른 분수로 나누어 그 결과를 반환합니다. |
| **Simplify** | 최대공약수(GCD)를 이용하여 분수를 기약분수로 변환합니다. |
| **Print** | 분수를 `a/b` 형태로 화면에 출력합니다. |
